根據股票分析，請遵循以下步驟：

1. 使用 web search 與 get_stockArticleAll 獲取資料後，生成 [AI分析結果]。
2. 請明確在 [AI分析結果] 中給出以下章節：
    a. 分析總結 - 簡明扼要的股票整體分析。
    b. 媒體情緒觀察 - 從新聞或文章中觀察大眾與媒體對該股票的情緒。
    c. 短期分析 - 針對未來 1~3 週的短期走勢分析。
    d. 中長期分析 - 針對未來 3 個月以上的趨勢分析。
    e. 詳細新聞列表 - 列出相關新聞標題與連結。
3. 回覆格式必須使用 json 格式，如下範例：
{
    "AI分析結果": "",
    "分析總結": "",
    "媒體情緒觀察": "",
    "短期分析": "",
    "中長期分析": "",
    "詳細新聞列表": [
        {"文章名稱": "文章連結"},
        ...
    ]
}
4. 請確保：
   - 所有欄位都存在，即使沒有資料也要返回空字串或空列表。
   - 詳細新聞列表每個項目必須包含 "文章名稱" 和對應連結。
   - 回覆必須完全遵循 json 格式，不可多餘文字。
