# AI-Stock

1.  **Activate the Virtual Environment:**
    ```bash
    uv init  # init porject (first create project after that no need call)
    uv sync  # install env
    source .venv/bin/activate  # On Linux/macOS
    .venv\Scripts\activate     # On Windows
    ```

2. **Run app**
    ```bash
    streamlit run main.py
    ```
